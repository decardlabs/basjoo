## 部署 v0.3.0 到服务器

### 服务器部署步骤
1. SSH 到 43.165.186.252
2. cd /opt/basjoo
3. git stash 暂存本地修改
4. git pull origin main (拉取 v0.3.0)
5. 修复 docker-compose.yml (添加缺失的端口映射 8000, 3008)
6. docker compose --profile prod up -d (重建容器)
7. 启动 ccbot-nginx

### 遇到的问题与解决
- **Nginx 无法启动** - 端口 80 被外部 Nginx 占用 → 移除 docker-compose nginx 端口映射
- **服务名不匹配** - locations.conf 引用 backend-prod/frontend-prod → 实际服务名正确，是网络问题
- **容器无网络连接** - Nginx 容器创建失败 → 强制重建解决
- **前端/后端端口未映射** - 新 docker-compose.yml 缺少端口映射 → 手动添加 `127.0.0.1:8000:8000` 和 `127.0.0.1:3008:3000`
